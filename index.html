Portfolio

Pollen

Pollen is an engaging and educational web application on forensic botany. Forensic botany is a niche field of forensic science that uses plants or plant parts in investigating criminal cases. It is not widely known to many, thus, it is an engaging educational tool for those who never heard of forensic botany. Additionally, it is a space for forensic botany enthusiasts to learn and discover cases they were not aware of. It was built with Heroku, Express, PHP, HTML and CSS. Hbnb

Hbnb is a console and data storage to replicate the back-end functionality of Airbnb. The console is a command-line interpreter that handles creating, displaying, destroying and updating user data. The data storage handles storing and updating user data. It was built entirely with Python and include unit test cases.

Monty

Monty was created to educate users about a stack data structure. It is a command-line interpreter that reads and executes Monty ByteCode files (.m extension files) that contain operation codes followed by the data to insert onto the stack. It was developed entirely with C and compiled with GCC.
